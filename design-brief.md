# CRA Construction Website Design Brief

## Project

CRA Construction Website

## Business context

CRA Construction is Cole Alberius's construction company and general contractor. The company works across custom homes, spec homes, residential development, light commercial construction, and infrastructure/site improvements.

## Target audiences

- Prospective custom-home clients
- Land/development partners
- Commercial/light-commercial clients
- Vendors/subcontractors checking legitimacy
- Lenders/investors/partners who need to see operational credibility
- Local Arkansas prospects who want a builder they can trust

## Brand posture

Premium but practical.

The website should feel more capable and established than a small builder template, but not overly glossy or fake. CRA should come across as a real operator: trusted, field-capable, organized, and experienced.

## Design principles

- Strong first impression
- Uncluttered navigation
- Real project photography when available
- Specific language over generic marketing copy
- Clear calls to action
- Mobile-first credibility
- Professional enough for commercial/development work
- Warm enough for custom residential clients

## Things to avoid

- Generic contractor stock-photo template feel
- Too many badges/icons/cards
- Overclaiming
- Fake stats unless Cole provides real numbers
- Busy hero sections
- Weak CTAs like “Learn More” everywhere
- Too much text above the fold

## Possible CTA language

- Start a Project
- Talk With CRA
- Request a Consultation
- View Our Work
- Discuss Your Build

## Possible homepage sections

1. Hero / positioning statement
2. Services overview
3. Featured projects / portfolio preview
4. Why CRA / operating principles
5. Development + construction capability
6. Process
7. Contact CTA

## Open items for Cole

We can proceed with assumptions, but these will improve quality:

- Does CRA already have a logo file and brand colors?
- Any existing website/domain?
- Best phone/email/contact destination?
- Do we have project photos?
- Which work type should the site prioritize first: custom homes, developments, light commercial, or site work?
- Should the tone lean luxury, local-trusted, or bold operator?
