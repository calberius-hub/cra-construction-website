# Pike Properties Reference — CRA Construction Website

Reference URL: https://pikeproperties.com

## Cole's direction

Cole wants the CRA Construction website modeled closely after Pike Properties' site direction, but CRA does **not** need these sections:

- Blog
- Foundation
- House plans / home plans

## What to borrow as design inspiration

Use Pike as a reference for structure, mood, and business posture, not as a literal copy.

Key ideas to adapt for CRA:

- Luxury custom-builder positioning
- Large photography-led homepage
- Simple navigation
- Clean current/previous project split
- Featured homes/projects as the core conversion content
- Direct contact CTA
- Social proof / Instagram-style project feed if CRA has photos available
- Minimal copy with confident project visuals doing most of the work

## What not to include

- No blog section
- No foundation section
- No house plans/home plans section
- No plan-purchase funnel
- No copied Pike copy, layouts, logos, images, or proprietary visual identity

## Proposed CRA navigation inspired by the reference

- Current Projects
- Previous Projects
- Custom Homes
- Developments
- About
- Contact

Possible simpler version:

- Current Projects
- Previous Work
- About CRA
- Contact

## Proposed homepage sections

1. Hero
   - Large project/home image
   - Short brand line
   - Primary CTA: `View Current Projects` or `Start a Project`

2. Current Projects
   - Featured active or upcoming CRA projects
   - Can include Rosemary Villas, current custom homes, spec homes, or development work if appropriate

3. Previous Work
   - Photo-led gallery of completed homes/projects

4. Built by CRA
   - Short credibility section explaining CRA's construction/development capability

5. Social / Field Updates
   - Optional Instagram-style feed only if photos/social content are available

6. Contact CTA
   - Simple form/contact block

## Copy posture

Short, direct, confident. Avoid generic contractor phrases.

Potential brand line directions:

- `Built with intention. Managed with discipline.`
- `Custom homes and development work across Central Arkansas.`
- `A construction company built for homes, communities, and long-term value.`
- `From custom homes to development infrastructure, CRA builds with ownership-level care.`

## Design posture

- Editorial luxury custom home builder
- Warm neutral palette
- Large imagery
- Minimal chrome
- Strong typography
- Calm spacing
- Premium but not flashy
